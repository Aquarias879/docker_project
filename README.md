# Docker Command

#build compose
```sh
$ docker-Compose build
```

#run docker compose
```sh
$ docker-compose up
```

#stop docker
```sh
$ docker-compose down
```

#access a running container
```sh
$ dokcer exec -it $service id bin/bash
```

#start only 1 service
```sh
$ docker start $service id
```

#close only 1 service
```sh
$ docker stop $service id
```
